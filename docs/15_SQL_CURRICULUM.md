# Purpose

This document defines the complete SQL curriculum used by ASCEND.

The objective is not to memorize SQL syntax.

The objective is to become capable of designing efficient relational databases, writing optimized queries, debugging production issues, and solving SQL interview questions confidently.

This curriculum is designed for backend engineers targeting high-paying product companies.

Estimated Duration

2 Months

Estimated Study Time

60 Hours
