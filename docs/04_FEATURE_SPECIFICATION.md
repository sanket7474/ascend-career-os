# Purpose

This document describes every feature that ASCEND will contain.

Every feature should provide measurable value to the user.

No feature should exist without helping career growth.

---

# FEATURE 1

## Dashboard

### Purpose

Provide a complete overview of the user's current state.

The dashboard should answer:

* What should I do today?
* How much have I improved?
* Am I on track?
* What is my next goal?

# Product Rule

If a feature does not help the user:

Learn better

Build better

Interview better

Earn better

or live better

it should not be added to ASCEND.

Keep the product focused.

Simple.

Powerful.

Actionable.
